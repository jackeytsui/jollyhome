## App Name

Jolly Home

## Subtitle

Your household operating system

## Short Description

Run shared living in one place: expenses, chores, shopping, meals, calendar, maintenance, rules, and household AI.

## Full Description

Jolly Home is built for people who live together and want one connected system instead of a pile of disconnected tools.

Use it to:

- split expenses and settle balances
- scan receipts and review extracted data
- manage chores and fairness
- plan meals and generate shopping lists
- track pantry stock and low-stock alerts
- coordinate calendars, guests, rules, and maintenance
- ask a household assistant grounded in real household state

Jolly Home is designed around one core idea: the home works better when money, food, timing, upkeep, and coordination are connected instead of scattered across different apps.
