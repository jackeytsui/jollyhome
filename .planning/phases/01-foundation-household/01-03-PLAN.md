export function useHousehold() {
  return {
    createHousehold,     // (name: string, avatarUrl?: string) => Promise<Household>

    switchHousehold,     // (householdId: string) => Promise<void>

  };
}

**createHousehold implementation:**
- Insert into `households` table: `{ name, avatar_url: avatarUrl, created_by: user.id }`
- Insert into `household_members`: `{ household_id: newHousehold.id, user_id: user.id, role: 'admin', status: 'active' }`
- Update `profiles` set `active_household_id = newHousehold.id`
- Update Zustand `householdStore`: `setActiveHousehold(id, name, 'admin')`
- Track analytics: `posthog.capture('household_created')`
- Return the created household

**loadActiveHousehold implementation:**
- Fetch profile's `active_household_id`
- If null, return (no active household)
- Fetch household data + membership role
- Fetch member count
- Update Zustand store

**switchHousehold implementation:**
- Update `profiles.active_household_id`
- Reload household data into store

2. Create `src/hooks/useInvite.ts`:

```typescript
export function useInvite() {
  return {
    createInvite,     // (householdId: string, options?: { expiryDays?: number, maxUses?: number }) => Promise<Invite>
    getInviteInfo,    // (token: string) => Promise<InviteInfo | null> — calls RPC
    redeemInvite,     // (token: string) => Promise<void>
    getInviteUrl,     // (token: string) => string
    listInvites,      // (householdId: string) => Promise<Invite[]>
    deleteInvite,     // (inviteId: string) => Promise<void>
    isLoading,
    error,
  };
}
```

**createInvite implementation:**
- Read household's `invite_expiry_days` to compute `expires_at`
- Insert into `household_invites`: `{ household_id, created_by: user.id, expires_at, max_uses }`
- Return the invite with token

**getInviteInfo implementation:**
- Call `supabase.rpc('get_household_invite', { invite_token: token })`
- Return the result which includes `household_name`, `member_count`, `is_valid`
- This works for unauthenticated users (SECURITY DEFINER function)

**redeemInvite implementation:**
- First call `getInviteInfo` to validate the invite
- Check `is_valid` — if false, throw appropriate error from i18n:
  - expired: `t('error.inviteExpired')` ("This invite has expired. Ask your household admin to send a new one.")
  - used up: `t('error.inviteUsed')` ("This invite link has already been used. Ask your household admin for a new one.")
- Check member limit: if `member_count >= FREE_TIER_MEMBER_LIMIT` and user is not paid, throw `t('error.memberLimit')`
- If `join_approval_required`: insert member with `status: 'pending'`
- Else: insert member with `status: 'active'`
- Increment `household_invites.use_count`
- Update `profiles.active_household_id` to the joined household
- Update Zustand store
- Track: `posthog.capture('household_joined')`

**getInviteUrl implementation:**
- `Linking.createURL(\`/invite/${token}\`)`
