**RLS Policies:**

profiles:
- SELECT: `auth.uid() = id` (own profile only) OR member of same household
- UPDATE: `auth.uid() = id` (own profile only)
- INSERT: `auth.uid() = id`

households:
- SELECT: user is active member (`id IN (SELECT household_id FROM household_members WHERE user_id = auth.uid() AND status = 'active')`)
- INSERT: authenticated users can create
- UPDATE: user is admin member

household_members:
- SELECT: user is active member of same household
- INSERT: handled by join function (not direct insert)
- UPDATE: admins of same household only
- DELETE: user can delete own membership OR admin can delete others

household_invites:
- SELECT: user is admin of the household
- INSERT: user is admin of the household
- DELETE: user is admin of the household

ai_credits:
- SELECT: `auth.uid() = user_id`
- UPDATE: `auth.uid() = user_id` (for incrementing usage)

**SECURITY DEFINER function** for invite lookup (unauthenticated users):
```sql
CREATE OR REPLACE FUNCTION public.get_household_invite(invite_token UUID)
RETURNS TABLE (
  id UUID,
  household_id UUID,
  household_name TEXT,
  household_avatar_url TEXT,
  member_count BIGINT,
  expires_at TIMESTAMPTZ,
  max_uses INTEGER,
  use_count INTEGER,
  is_valid BOOLEAN
)
LANGUAGE sql
SECURITY DEFINER
SET search_path TO 'public'
AS $$
  SELECT
    hi.id,
    hi.household_id,
    h.name AS household_name,
    h.avatar_url AS household_avatar_url,
    (SELECT COUNT(*) FROM household_members hm WHERE hm.household_id = hi.household_id AND hm.status = 'active') AS member_count,
    hi.expires_at,
    hi.max_uses,
    hi.use_count,
    (hi.expires_at IS NULL OR hi.expires_at > now()) AND (hi.max_uses IS NULL OR hi.use_count < hi.max_uses) AS is_valid
  FROM household_invites hi
  JOIN households h ON h.id = hi.household_id
  WHERE hi.token = invite_token;
$$;
```

**Trigger** for auto-creating profile on signup:
```sql
CREATE OR REPLACE FUNCTION public.handle_new_user()
RETURNS trigger
LANGUAGE plpgsql
SECURITY DEFINER SET search_path = public
AS $$
BEGIN
  INSERT INTO public.profiles (id, display_name)
  VALUES (new.id, COALESCE(new.raw_user_meta_data->>'full_name', new.raw_user_meta_data->>'name', split_part(new.email, '@', 1)));
  RETURN new;
END;
$$;

CREATE TRIGGER on_auth_user_created
  AFTER INSERT ON auth.users
  FOR EACH ROW EXECUTE FUNCTION public.handle_new_user();
```

BEGIN
  NEW.updated_at = now();

END;
