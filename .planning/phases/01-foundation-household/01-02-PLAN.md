1. Create `src/hooks/useAuth.ts` — a custom hook wrapping all Supabase auth operations:

```typescript
export function useAuth() {
  return {
    signUpWithEmail,      // (email, password, displayName) => Promise
    signInWithEmail,      // (email, password) => Promise
    signInWithMagicLink,  // (email) => Promise
    signInWithGoogle,     // () => Promise
    signInWithApple,      // () => Promise
    resetPassword,        // (email) => Promise
    signOut,              // () => Promise
    resendVerification,   // () => Promise
    isLoading,            // boolean
    error,                // string | null
    clearError,           // () => void
  };
}
```

Implementation details:

**signUpWithEmail**: Call `supabase.auth.signUp({ email, password, options: { data: { display_name: displayName } } })`. On success, the profile trigger in the DB auto-creates the profile row. Return the user object. Set `emailConfirmationRequired: true` state if email is not confirmed.

**signInWithEmail**: Call `supabase.auth.signInWithPassword({ email, password })`. On error, map Supabase error messages to user-friendly strings from i18n:
- `"Invalid login credentials"` -> `t('error.authFailure')` ("Incorrect email or password. Please try again or reset your password.")
- Network errors -> `t('error.network')` ("Connection problem. Check your internet and try again.")

**signInWithMagicLink**: Call `supabase.auth.signInWithOtp({ email, options: { emailRedirectTo: Linking.createURL('/') } })`. Return success flag (email sent).

**signInWithGoogle**: Follow RESEARCH.md Pattern 2 exactly:
```typescript
import * as WebBrowser from 'expo-web-browser';
import * as AuthSession from 'expo-auth-session';

WebBrowser.maybeCompleteAuthSession();

const redirectTo = AuthSession.makeRedirectUri();
const { data, error } = await supabase.auth.signInWithOAuth({
  provider: 'google',
  options: { redirectTo, skipBrowserRedirect: true },
});
if (data.url) {
  const result = await WebBrowser.openAuthSessionAsync(data.url, redirectTo);
  if (result.type === 'success') {
    await supabase.auth.exchangeCodeForSession(result.url);
  }
}
```

**signInWithApple**: Identical pattern to Google but with `provider: 'apple'`.

**resetPassword**: Call `supabase.auth.resetPasswordForEmail(email, { redirectTo: Linking.createURL('/reset-password') })`.

**resendVerification**: Call `supabase.auth.resend({ type: 'signup', email })`.

**signOut**: Call `supabase.auth.signOut()`.

All functions wrap calls in try/catch and set `error` state on failure. Track `isLoading` state per operation.
