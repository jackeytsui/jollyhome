    **Create `src/hooks/useExpenses.ts`:**

    Follow the exact pattern from `useMembers.ts` — local `useState` for data, loading, error. No React Query.

    ```typescript
    import { useState, useEffect, useCallback } from 'react';
    import { supabase } from '@/lib/supabase';
    import { useHouseholdStore } from '@/stores/household';
    import { useExpenseStore } from '@/stores/expenses';
    import type { Expense, ExpenseSplit, CreateExpenseInput, SplitPreset } from '@/types/expenses';
    ```

    Hook returns: `{ expenses, loading, error, createExpense, loadExpenses, presets, loadPresets }`

    `loadExpenses()`:
    - Get `activeHouseholdId` from `useHouseholdStore`
    - Query `supabase.from('expenses').select('*, expense_splits(*)').eq('household_id', householdId).is('deleted_at', null).order('expense_date', { ascending: false }).limit(50)`
    - Set state with data
    - If offline (error), do NOT overwrite existing cached state

    `createExpense(input: CreateExpenseInput)`:
    - Check network connectivity. If offline, call `useExpenseStore.getState().enqueue(input)` and return
    - Call `supabase.rpc('create_expense', { p_household_id: input.household_id, p_description: input.description, p_amount_cents: input.amount_cents, p_category: input.category, p_paid_by: input.paid_by, p_split_type: input.split_type, p_splits: JSON.stringify(input.splits), p_tax_cents: input.tax_cents ?? 0, p_tip_cents: input.tip_cents ?? 0, p_is_private: input.is_private ?? false, p_receipt_url: input.receipt_url ?? null, p_expense_date: input.expense_date ?? new Date().toISOString().split('T')[0] })`
    - On success, call `loadExpenses()` to refresh list

    `loadPresets()`:
    - Query `supabase.from('split_presets').select('*').eq('household_id', householdId)`
    - Store in `presets` state

    Supabase Realtime subscription in `useEffect`:
    - Subscribe to `supabase.channel('household:${householdId}:expenses').on('postgres_changes', { event: '*', schema: 'public', table: 'expenses', filter: 'household_id=eq.${householdId}' }, () => { loadExpenses(); })`
    - Cleanup: `supabase.removeChannel(channel)` in return function
    - Dependency array: `[householdId]`

    **Create `src/hooks/useBalances.ts`:**

    ```typescript
    import { useState, useEffect, useMemo } from 'react';
    import { supabase } from '@/lib/supabase';
    import { useHouseholdStore } from '@/stores/household';
    import { computeBalances, simplifyDebts } from '@/lib/expenseMath';
    import type { LedgerEntry, Balance, Settlement } from '@/types/expenses';
    ```

    Hook returns: `{ netBalances, simplifiedDebts, settlements, loading, loadBalances, loadSettlements }`

    `loadBalances()`:
    - Fetch all non-deleted expenses with splits for household
    - Transform to `LedgerEntry[]` (paidBy + splits array)
    - Also fetch all settlements for household
    - Compute `net = computeBalances(entries)` then subtract settlements from net positions
    - Compute `simplified = simplifyDebts(adjustedNet)`
    - Set `netBalances` and `simplifiedDebts` state

    `loadSettlements()`:
    - Query `supabase.from('settlements').select('*').eq('household_id', householdId).order('created_at', { ascending: false })`

    Realtime: subscribe to `settlements` table changes on same channel pattern, refresh on change.

    Both hooks call their load functions in `useEffect` on mount when `householdId` is truthy.
