    **`supabase/functions/process-receipt/index.ts`:**

    Create Deno Edge Function (Supabase Edge Functions use Deno runtime):

    ```typescript
    import { serve } from 'https://deno.land/std@0.168.0/http/server.ts';
    import { createClient } from 'https://esm.sh/@supabase/supabase-js@2';
    ```

    Handler:
    - Parse request body: `{ storage_paths: string[], household_id: string, member_names: string[] }`
    - `storage_paths` is an array (supports multi-page receipts)
    - Create Supabase client with service role key for storage access
    - For each storage_path: generate signed URL via `supabase.storage.from('receipts').createSignedUrl(path, 60)`
    - Download images as base64
    - Call OpenAI API directly (no SDK — use fetch to `https://api.openai.com/v1/chat/completions`):
      - Model: `gpt-4o`
      - Messages: system prompt (receipt parsing assistant) + user message with image(s) as base64 content parts
      - System prompt exact text:
        ```
        You are a receipt parsing assistant. Extract structured data from the receipt image(s).
        Return JSON exactly matching this schema:
        {
          "store_name": string,
          "date": "YYYY-MM-DD" | null,
          "items": [{ "name": string, "price_cents": integer, "classification": "shared"|"personal", "suggested_owner": string|null }],
          "subtotal_cents": integer,
          "tax_cents": integer,
          "tip_cents": integer,
          "total_cents": integer
        }
        Household members for classification hints: ${member_names.join(', ')}
        Classify items as "personal" if they are clearly individual (e.g., specific medications, personal care items with a single person's name).
        Default to "shared" for food, household supplies, and anything ambiguous.
        All prices must be in cents (integer). $4.99 = 499.
        If multiple images are provided, they are pages of the same receipt — combine items from all pages.
        ```
      - max_tokens: 4096
      - temperature: 0 (deterministic)
      - response_format: `{ type: "json_object" }` to enforce JSON output
    - Parse OpenAI response JSON
    - **Validate response shape** (Pitfall 7): ensure all required fields exist, fill defaults for missing (`tax_cents: 0`, `tip_cents: 0`, `date: null`). Ensure all `price_cents` are integers.
    - Return validated JSON with CORS headers

    Environment variables needed: `OPENAI_API_KEY` (from Edge Function secrets)

    Error handling: if OpenAI call fails or returns invalid JSON, return `{ error: "Failed to process receipt", details: errorMessage }` with 422 status.

    **`src/hooks/useReceipt.ts`:**

    ```typescript
    import { useState, useCallback } from 'react';
    import { supabase } from '@/lib/supabase';
    import { useHouseholdStore } from '@/stores/household';
    import { useAuthStore } from '@/stores/auth';
    ```

    Interface `ReceiptData`:
    ```typescript
    interface ReceiptItem { name: string; price_cents: number; classification: 'shared' | 'personal'; suggested_owner: string | null; }
    interface ReceiptData { store_name: string; date: string | null; items: ReceiptItem[]; subtotal_cents: number; tax_cents: number; tip_cents: number; total_cents: number; }
    ```

    Returns: `{ receiptData, images, loading, error, captureImage, pickFromGallery, addPage, removePage, processReceipt, clearReceipt }`

    `captureImage(uri: string)`:
    - Add URI to `images` array state
    - For single page: set images to [uri]. For multi-page: append.

    `pickFromGallery()`:
    - Call `ImagePicker.launchImageLibraryAsync({ mediaTypes: ImagePicker.MediaTypeOptions.Images, quality: 0.8 })`
    - If not canceled, add result.assets[0].uri to images

    `addPage(uri: string)`:
    - Append to images array

    `removePage(index: number)`:
    - Remove image at index

    `processReceipt()`:
    - Set loading=true
    - For each image URI: upload to Supabase Storage bucket 'receipts' under path `receipts/${householdId}/${uuid}.jpg`
      - Read file via `fetch(uri)` then `blob`
      - Upload via `supabase.storage.from('receipts').upload(path, blob, { contentType: 'image/jpeg' })`
    - Collect storage_paths array
    - Get member names from useMembers or household store
    - Call Edge Function: `supabase.functions.invoke('process-receipt', { body: { storage_paths, household_id: householdId, member_names } })`
    - Parse response data into ReceiptData
    - Set receiptData state
    - Set loading=false
    - If error, set error state with user-friendly message

    `clearReceipt()`:
    - Reset all state (images, receiptData, error)

    Total processing time target: < 4 seconds (upload + OCR)
